# Todolist Backend 

## Get Started

- Clone repository
  ```
  git clone 
  ```

- Install dependencies
  ```
  npm install
  ```

- Setup database server environment
  - Creating `.env` file in form of `.env.sample`
  - Initialize database 
    - Run mysql
      ```
      mysql -u <username> -p
      ```
    - Run `init.sql` 
      ```
      SOURCE ./
      ```

- Start app
  ```
  npm start
  ```
